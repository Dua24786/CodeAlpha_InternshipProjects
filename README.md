# Project Tasks

## ✅ Task 1: Student Grade Tracker 📚

### Description
The Student Grade Tracker is a Java application designed to manage student academic records. It allows users to enter student grades, calculate average marks, identify the highest and lowest scores, and generate a summary report of student performance.

### Features
- Add and manage student grades
- Calculate average marks
- Find highest and lowest scores
- Generate performance summary
- Store data using Arrays or ArrayLists
- Console-based or GUI-based interface

### Concepts Used
- Java Fundamentals
- Object-Oriented Programming (OOP)
- Arrays & ArrayLists
- Loops and Conditional Statements
- Methods

## ✅ Task 2: Stock Trading Platform 📊

### Description
The Stock Trading Platform is a Java-based application that simulates a basic stock market environment. Users can buy and sell stocks, monitor portfolio performance, and manage trading transactions.

### Features
- Buy and sell stocks
- Display market data
- Portfolio performance tracking
- Transaction management
- Object-Oriented design
- File handling for data storage (optional)

### Concepts Used
- Java Fundamentals
- Object-Oriented Programming (OOP)
- Collections
- File Handling
- Exception Handling

## ✅ Task 3: Hotel Reservation System 🏨

### Description
The Hotel Reservation System is a Java application that allows users to search, book, and manage hotel room reservations. The system manages customer details, room availability, reservations, and booking records.

### Features
- Search available rooms
- Book hotel rooms
- Cancel reservations
- Room categorization (Standard, Deluxe, Suite)
- Booking details management
- Payment simulation
- Store booking records using File Handling or Database

### Concepts Used
- Java Fundamentals
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
- Collections
